app连接shizuku服务，执行shell命令
===============================

注意：

1. 使用前请确保已经安装了`Shizuku`并授予了`shell`权限
2. 并不能执行所有`shell`命令，如`cd`
3. 当执行的命令中，路径带有空格时，使用双引号包裹，如`ls "/temp dir/"`
